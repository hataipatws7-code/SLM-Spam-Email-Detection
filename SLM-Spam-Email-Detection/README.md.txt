# Small Language Model for Spam Email Detection

This project compares multiple Transformer-based models for spam vs ham classification.

## Models
- DistilBERT
- BERT-base
- ALBERT
- MiniLM

## Evaluation
- Accuracy
- Precision
- Recall
- F1-score
- Training Time
- Inference Time
- GPU Memory

## Dataset
Spam-Ham combined dataset (~82k emails)
