# Small Language Model for Spam Email Detection

This project evaluates multiple Transformer-based models for spam vs ham classification.

## Models Compared
- DistilBERT
- BERT-base
- ALBERT
- MiniLM

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Training Time
- Inference Time
- GPU Memory Usage

## Project Structure
notebooks/        -> Training experiments
results/          -> Model evaluation outputs
requirements.txt  -> Required libraries
